Analysis client

Analyse a set of steps of a run.

Considering the example run:

run1/
    0_topoaa/
    1_rigidbody/
    2_seletop/
    3_flexref/
    (etc...)

USAGE:

haddock3-analyse -r <run_dir> -m <num_modules>
haddock3-analyse -r run1 -m 1 3
Where, -m 1 3 means that the analysis will be performed on 1_rigidbody

and 3_flexref.
