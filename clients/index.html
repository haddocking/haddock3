Command-line interfacesÔÉÅ

HADDOCK3 has a series of command-line interfaces (CLIs), from which the most important one is haddock3. You can ask help to the CLIs with the -h flag:

haddock3 -h

The available CLIs are the following:

  • Run HADDOCK3
  • Retrieve config
  • PDB preprocessing client
  • Copy steps to new run
  • Clean output Client
  • Unpack output
  • HADDOCK3 MPI Run
  • Benchmark Client
  • Benchmark Daemon
