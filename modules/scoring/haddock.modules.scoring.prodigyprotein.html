Module contents

PRODIGY protein complex scoring module

This module performs the scoring of input complexes using PRODIGY (protein). It predicts deltaG of the complex and can return predictions as either deltaG or pKd values.

Note that PRODIGY-protein is limited to natural amino-acids.
