Parameters helper

Relates to logic or definition of parameters.

haddock.gear.parameters.MANDATORY_YAML = PosixPath('/opt/hostedtoolcache/Python/3.10.16/x64/lib/python3.10/site-packages/haddock/core/mandatory.yaml')

The mandatory general arguments of the configuration file.

haddock.gear.parameters.OPTIONAL_YAML = PosixPath('/opt/hostedtoolcache/Python/3.10.16/x64/lib/python3.10/site-packages/haddock/core/optional.yaml')

The optional general arguments of the configuration file.
